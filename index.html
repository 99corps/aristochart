    Aristochart is highly customizable and flexible line charting library for canvas. Aristochart allows you to focus on aesthetic while it manages the data behind the scenes.

    Themes

    Aristochart has a growing list of community sourced themes for you to pick and choose from.

    new Aristochart(document.getElementById("holder"), {
	data: {
		x: 10,
		y: [0, 1, 2, 3, 4, 5, 6, 6, 5, 4, 3, 3, 4, 5, 6, 7, 8, 9, 10],
		y1: [10, 9, 8, 7, 7, 6, 2, 1, 1, 1],
		y2: [4, 4, 4, 3, 3, 2, 2, 1, 0, 0, 0, 0]
	}
}, theme);

    API

    Aristochart's API was designed to be as simple as possible to allow you to focus on the aesthetic and presentation.

    Documentation

    Aristochart has indepth documentation of it's internals and options that you can edit and extend. All packaged together and works locally.

    Examples

    Aristochart has a growing set of examples and demonstrations with their code to let you see it in action and help you learn by doing.

    Open-source

    Aristochart is completely open-source and hosted on Github. Fork the repo and start contributing on your own features and themes!
